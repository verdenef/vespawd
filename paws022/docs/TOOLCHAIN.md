# Toolchain reference (POS v1.1.7)

POS is **tool-neutral**. You choose planner, executor, and documenter in setup.

## Roles

| Role | Does | Typical tools |
|------|------|----------------|
| **Planner** | Produces `POS MASTER PROMPT` | Gemini, ChatGPT, Claude, manual |
| **UI designer** (optional) | Fills `design/DESIGN.md` — [UI_DESIGN.md](UI_DESIGN.md) | Gemini, ChatGPT, manual, or skip |
| **Executor** | Writes `tasks/`, `design/` exports, code in `src/`, HANDOFF | Cursor, VS Code+Copilot, Windsurf, JetBrains AI |
| **Documenter** | School report from HANDOFF + rubric | Gemini, ChatGPT, manual |

**UI tools (executor-side, optional):** Google Stitch MCP in Cursor — [STITCH_CURSOR.md](STITCH_CURSOR.md). Also Figma, v0, Penpot — anything that lands artifacts in `design/`.

## Common stacks

| Planner | Executor | Documenter | POS files to use |
|---------|----------|------------|------------------|
| Gemini | Cursor | Gemini | `EXTERNAL_AGENTS_SETUP.md`, `.cursor/rules/pos.mdc`, `executor_rules.md` |
| Gemini (Gem) | VS Code + Copilot | Gemini (Gem) | Copy `executor_rules.md` into project instructions |
| ChatGPT GPT | Cursor | ChatGPT | Custom GPT instructions from `planner_prompt.md` shape |
| Manual | Any | Manual | Write `tasks/current_task.md` yourself |

## Executor setup by tool

| Executor | What to enable |
|----------|----------------|
| **Cursor** | Open app repo; `.cursor/rules/pos.mdc` applies automatically |
| **VS Code + Copilot** | Add `AGENTS.md` / paste [.ai/executor_rules.md](../.ai/executor_rules.md) to project instructions |
| **Windsurf** | Same as VS Code — project rules + `executor_rules.md` |
| **Other** | `AGENTS.md` + `executor_rules.md` |

## Planner setup by tool

| Planner | Setup |
|---------|--------|
| **Gemini** | Copy prompts into Instructions; optional Gem **Knowledge** files — [EXTERNAL_AGENTS_SETUP.md](EXTERNAL_AGENTS_SETUP.md) § Gemini Knowledge |
| **ChatGPT GPT** | Create GPT; use OUTPUT CONTRACT and format from [.ai/planner_prompt.md](../.ai/planner_prompt.md) |
| **Claude Project** | Project instructions with same POS MASTER PROMPT format |
| **Manual** | Skip planner; write tasks yourself |

## Privacy

Configure **your own** external agents/GPTs. Share the **repo**, not personal assistant links. Instructions text lives in this repository (`instructionsVersion`).

## Config

Stored in `%USERPROFILE%\.pos\config.json` (see `pos.config.json.example`). Generated by **`Setup-POS.bat`** / `pos-setup.ps1` (wizard HTML is layout guide only).
