# Completed Tasks (POS Scheduler)

Immutable log of finished work. **Do not** use this folder for active tasks.

## Filename Convention

```
YYYY-MM-DD-short-slug.md
```

Example: `2026-05-22-add-user-auth.md`

## File Contents

Each file should include:

- Goal and acceptance criteria (from `current_task.md`)
- Summary of what changed
- How to verify
- Links to PRs/commits if applicable

When this directory has real entries, you may remove `.gitkeep`.
