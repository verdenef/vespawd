# Vedaws

Vedaws is a domain-neutral Development Operating System (DevOS) that orchestrates project state, workflows, workers, automation, and AI capabilities through a plugin-first architecture.

## Start here

- Bootstrap and onboarding: [`VEDAWS_BOOTSTRAP.md`](VEDAWS_BOOTSTRAP.md)
- Architecture index: [`design/README.md`](design/README.md)
- Frozen API surface: [`docs/API_STABILITY.md`](docs/API_STABILITY.md)

## Quick install

```bash
pip install -e ".[dev]"
vedaws version
vedaws --help
```
